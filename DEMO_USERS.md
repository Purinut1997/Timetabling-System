## 🔧 การเข้าสู่ระบบ

### ขั้นตอนการเข้าสู่ระบบ:
1. เปิดหน้าเว็บ: `http://localhost:3000`
2. คลิกปุ่ม **"เข้าสู่ระบบ"**
3. กรอก **Email** และ **Password** ตามบทบาทที่ต้องการ
4. คลิกปุ่ม **"เข้าสู่ระบบ"**
5. ระบบจะนำทางไปยัง Dashboard ตามบทบาท

---

## ⚠️ หมายเหตุ

- รหัสผ่านเหล่านี้เป็น **รหัสผ่านตัวอย่าง** สำหรับการทดสอบระบบเท่านั้น
- ในระบบจริงควรใช้รหัสผ่านที่ **ปลอดภัยและซับซ้อน** กว่านี้
- ไม่ควรใช้รหัสผ่านตัวอย่างในระบบ production

## 🚀 เริ่มต้นใช้งาน

1. รันคำสั่ง: `npm run dev`
2. เปิดเบราว์เซอร์ที่: `http://localhost:3000`
3. เลือกบทบาทที่ต้องการทดสอบ
4. เข้าสู่ระบบด้วยข้อมูลด้านบน
5. เริ่มทดสอบฟังก์ชันต่างๆ ในระบบ
